Build Your First Website with AI

Use Claude or ChatGPT to create and deploy a real website. No coding experience required.

Beginner ~2 hours

What you'll learn

  • How to write prompts that get AI to generate good code
  • Basic HTML and CSS concepts (enough to understand what AI gives you)
  • How to test your website locally
  • How to deploy for free so anyone can visit it

Part 1: Planning

1
Pick something to build

Start simple. Good first projects:

  • Personal site - about you, your interests, links to social media
  • Portfolio - showcase work or projects
  • Landing page - promote an idea or product
2
Sketch it out

Grab paper and draw a rough layout. Nothing fancy - just boxes showing where things go:

  • Header (your name/logo)
  • Main content sections
  • Footer

This gives you something concrete to describe to the AI.

Part 2: Writing the prompt

3
Be specific

Vague prompts get vague results. Instead of "make me a website", describe exactly what you want:

Create a personal portfolio website with: - Header with my name "Alex Chen" and nav links - Hero section with a short intro - About section with a few paragraphs - Projects section with 3 cards (use placeholders) - Footer with email and social links Dark theme, purple accents. Single HTML file with CSS included. Make it work on mobile.
Why this works

You've told AI exactly what sections to include, what color scheme to use, and the technical constraints. Less guessing = better output.

Understanding what you get back

AI will give you HTML and CSS code. Quick overview:

  • HTML - the structure (text, links, images)
  • CSS - the styling (colors, fonts, layout)
  • <head> - page settings and styles
  • <body> - everything visible on the page

You don't need to understand every line. You just need to know enough to make tweaks.

Part 3: Testing

4
Save and open
  1. Copy the code AI gave you
  2. Paste into a text editor
  3. Save as index.html
  4. Double-click the file to open in browser
5
Make changes

Something look off? Ask AI to fix it:

  • "Make the header sticky"
  • "Change the purple to blue"
  • "Add more space between sections"
  • "Make the font bigger on mobile"

Copy the updated code, save, refresh your browser.
