Learn how to create a complete website from scratch using AI tools. No prior coding experience required - AI will help you every step of the way!

Part 1: Planning Your Website

1
Decide What to Build

Before writing any code, think about what you want your website to do. Here are some beginner-friendly ideas:

  • Personal Portfolio - Showcase your work and skills
  • About Me Page - Tell visitors about yourself
  • Simple Blog - Share your thoughts and ideas
  • Landing Page - Promote a project or idea
2
Sketch Your Design

Draw a simple sketch of how you want your website to look. Include:

  • Header with your logo/name
  • Navigation menu
  • Main content area
  • Footer with contact info

This doesn't need to be pretty - just a rough layout to guide the AI!

üí° Pro Tip

The more specific you are with AI, the better results you'll get. Instead of "make me a website", try "create a personal portfolio website with a dark theme, hero section, about section, and contact form".

Part 2: Writing Your First Prompt

3
Craft Your AI Prompt

Here's an example prompt you can customize:

Create a simple personal portfolio website with:

- A header with my name "Your Name" and navigation links
- A hero section with a welcome message
- An "About Me" section with a brief bio
- A "Projects" section with 3 placeholder project cards
- A footer with social media links

Use a modern dark theme with purple accents.
Make it responsive for mobile devices.
Put all the code in a single HTML file with embedded CSS.

Understanding the Response

The AI will generate HTML and CSS code for you. Don't worry if you don't understand every line - that's normal! Here's what the main parts mean:

  • HTML - The structure and content of your page (text, images, links)
  • CSS - The styling (colors, fonts, spacing, layouts)
  • <head> - Contains page title and styling
  • <body> - Contains everything you see on the page

Part 3: Testing Your Website

4
Save and Open Your File
  1. Copy the code from the AI response
  2. Open a text editor (VS Code, Notepad, etc.)
  3. Paste the code
  4. Save the file as index.html
  5. Double-click the file to open it in your browser
5
Make Changes with AI

Want to change something? Just ask the AI! Example prompts:

  • "Change the background color to dark blue"
  • "Make the font bigger"
  • "Add a contact form"
  • "Make the project cards side by side"
