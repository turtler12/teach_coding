TrustAI Coding Exercises

10 exercises to master block-based programming

1 Hello World Beginner
Goal: Get familiar with adding blocks and running code.

Add a Print block and make it display "Hello, World!"

Expected Output:
Hello, World!
2 Create and Print a Variable Beginner
Goal: Learn how variables store values.
  1. Add a Create variable block and set name to age and value to 12
  2. Add a Print variable block to display age
Expected Output:
12
3 Simple Math Beginner
Goal: Practice changing variable values.
  1. Create a variable called score with value 10
  2. Use Change by to add 5 to score
  3. Print score
Expected Output:
15
4 Repeat a Message Easy
Goal: Introduction to loops.

Use a Repeat times block to print "I love coding!" exactly 3 times.

Expected Output:
I love coding! I love coding! I love coding!
5 Counting Up Easy
Goal: Combine loops with variables.
  1. Create a variable count with value 0
  2. Use a Repeat times block (5 times)
  3. Inside the loop: Change by 1 to count, then Print variable count
Expected Output:
1 2 3 4 5
6 Making Decisions Medium
Goal: Introduction to if statements.
  1. Create a variable temperature with value 35
  2. Add an If block with condition temperature > 30
  3. Inside the if: Print "It's hot outside!"
Expected Output:
It's hot outside!
7 If-Else Choices Medium
Goal: Learn branching with if-else.
  1. Create a variable grade with value 85
  2. Add an If-Else block with condition grade >= 70
  3. In the if section: Print "You passed!"
  4. In the else section: Print "Keep trying!"
Expected Output:
You passed!
Challenge: Change grade to 50 and run again. What happens?
8 Multiplication Table Medium
Goal: Combine loops and math operations.

Create a program that prints the 5 times table (5 x 1 through 5 x 5):

  1. Create variable result with value 0
  2. Create variable multiplier with value 0
  3. Use Repeat times (5 times)
  4. Inside: Change multiplier by 1, set result to multiplier * 5 using Set variable, then print result
Expected Output:
5 10 15 20 25
9 Sum of Numbers Hard
Goal: Use loops to accumulate a total.

Calculate the sum of numbers from 1 to 10:

  1. Create total with value 0
  2. Create number with value 0
  3. Repeat times 10 times:
  4. Change number by 1
  5. Change total by the current value of number
  6. After the loop, print total
Expected Output:
55
10 FizzBuzz (Simplified) Hard
Goal: Combine loops with multiple conditions.

Print numbers 1-10, but:

  • If the number equals 5, print "FIVE!" instead
  • If the number equals 10, print "TEN!" instead
  1. Create num with value 0
  2. Repeat times 10 times:
  3. Change num by 1
  4. Use If-Else with condition num == 5
  5. If true: Print "FIVE!"
  6. Else: Use another If-Else with condition num == 10
  7. If true: Print "TEN!"
  8. Else: Print variable num
Expected Output:
1 2 3 4 FIVE! 6 7 8 9 TEN!

20 Reverse a List Expert

Create a list [1, 2, 3, 4, 5] and build a new reversed list [5, 4, 3, 2, 1].

  1. Create list original with items 1, 2, 3, 4, 5
  2. Create empty list reversed
  3. Create variable i with value 5
  4. Repeat times 5 times:

  6. Use Get item at index i from original, store in item
  7. Add item to list reversed
  8. Print variable reversed
Expected Output:
[5, 4, 3, 2, 1]

Tips for Success

  • Click on a block in the left panel to add it to your workspace
  • To add blocks inside a loop or if statement, click the indented area first (you'll see a dashed outline), then click the block you want to add
  • Use the drag handles (⋮⋮) to reorder blocks
  • Click "Run Code" to see your output
  • Click "Clear All" to start fresh
  • Check the Variables panel to see current values
  • Don't forget the quotes around text: "Hello" not Hello
